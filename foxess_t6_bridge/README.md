# FoxESS HA Add-ons

Repozytorium dodatków Home Assistanta dla falowników FoxESS T-series ver 3.47/3.01/0.01

## Dostępne add-ony

- **FoxESS T6 Bridge** — lokalny bridge EE11A -> MQTT -> Home Assistant dla FoxESS T-series.

## Instalacja

W Home Assistant:
1. Settings -> Apps / Add-on Store
2. Dodaj URL tego repozytorium
3. Zainstaluj `FoxESS T6 Bridge`

## Wymagania

- Home Assistant z obsługą add-onów
- Broker MQTT (np. Mosquitto)
- Konwerter Elfin EE11A / EE11A LAN
- FoxESS T-series
